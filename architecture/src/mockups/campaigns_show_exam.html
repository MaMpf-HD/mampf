Policy Phases Explained
  • Registration: Checked when students try to register. Blocks registration if not satisfied. Use for hard requirements like email domain.
  • Finalization: Checked before materializing roster. Prevents completion if any confirmed student fails. Standard for lecture performance: students can register freely, certification verified at finalization.
